🌐 Network Security Portfolio

This is my personal portfolio website showcasing my journey as an aspiring Network Security Engineer. It highlights my skills, certifications, projects, and career roadmap in cybersecurity and networking.

---

🚀 Built with React + Tailwind CSS, featuring:

Dark/Light Mode Toggle 🌙☀️

Animated CLI simulations 💻

VLAN Network visualization 🎛️

Professional Contact Form ✉️

Responsive design for all devices 📱💻
